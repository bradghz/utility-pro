UTILITY CHILD THEME
http://www.carriedils.com/themes/utility/

REQUIREMENTS
This theme requires the Genesis Framework (The Genesis parent theme needs to be in the wp-content/themes/ directory, but does not need to be activated.)
This theme has HTML5 enabled by default and therefore only requires WordPress 3.6 or greater.

INSTALL
1. Upload the Utility theme folder via FTP to your wp-content/themes/ directory. 
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Utility theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking. To set up the theme like the demo, please visit http://www.carriedils.com/themes/utility/.

WIDGET AREAS
Primary Sidebar - This is the primary sidebar if you are using the Content/Sidebar or Sidebar/Content Site Layout option.
Utility Bar Left - This widgetized area appears above the header and to the left.
Utility Bar Right - This widgetized area appears above the header and to the right.
Header Right - This is the widgetized area that appears at the top right of the header. If it is active then Genesis style will follow the Partial Header style otherwise it will follow the Full Image Header Style, if image header is selected.
Home Welcome - This is the welcome area at the top of the home page.
Home Gallery- This is the gallery area at the middle of the home page.
Call to Action - This is the call to action area at the bottom of the home page.
Footer #1 - This is the first column of the footer section.
Footer #2 - This is the second column of the footer section.
Footer #3 - This is the third column of the footer section.

PLUGINS USED IN DEMO
To set the theme up per the demo, you will need to install the following plugins:

* Genesis E-News Extended by Brandon Kraft (http://wordpress.org/plugins/genesis-enews-extended/)
* Gravity Forms (premium plugin)
